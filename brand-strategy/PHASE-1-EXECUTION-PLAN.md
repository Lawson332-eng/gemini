## 📅 Month 1-3 详细执行计划

### Week 1-2: 竞品深度研究 + 供应商初筛

#### 竞品分析（必做功课）
**目标品牌**：
1. **Glamnetic** - 你的北极星对标
2. **Static Nails** - 高端可重复使用
3. **KISS** - 平价大众市场
4. **Olive & June** - 沙龙品质DTC
5. **imPRESS** (Broadway Nails) - CVS/Target渠道

**分析维度**（创建表格对比）：
- [ ] 价格带（单套售价、套装价格、shipping费用）
- [ ] 产品规格（片数、尺寸选择、附带工具）
- [ ] 材质（ABS vs gel-based vs 其他）
- [ ] 设计风格（nude/French/artistic/seasonal）
- [ ] 包装（盒子材质、开箱体验、说明书）
- [ ] 复用次数宣称（1次 vs 5次 vs 10次+）
- [ ] 社交媒体策略（TikTok内容类型、posting频率）
- [ ] 评论分析（Amazon/官网的top complaints）

**行动步骤**：
```
1. 购买样品（预算$200-300）：
   - Glamnetic: 3套不同设计（nude, French, artistic）
   - Static Nails: 2套（研究复用系统）
   - KISS: 2套平价款（成本基准）

2. 实测对比：
   - 佩戴体验（贴合度、压力感）
   - 粘合系统（adhesive tabs vs glue）
   - 耐用性（48小时测试）
   - 拆卸便利性
   - 复用可能性

3. 拆解包装：
   - 拍照记录每个细节
   - 测量尺寸
   - 材料推测
```

### Week 3-4: 打样与质量对标

**目标**：从5-8家中选出3家同时打样

**打样策略**：
1. **设计选择**：选2个最保险的设计打样
   - Design A: Nude/Natural（市场最大，失败率最低）
   - Design B: French Tips（经典常青款）

2. **质量检查清单**（对比Glamnetic样品）：
   ```
   材料质量：
   - [ ] 表面光泽度
   - [ ] 透明度/色彩饱和度
   - [ ] 厚度一致性（0.3-0.5mm标准）
   - [ ] 边缘打磨平滑度

   贴合系统：
   - [ ] 甲片弧度（C-curve自然度）
   - [ ] 尺寸准确性（10-12个尺寸）
   - [ ] 粘合材料质量（adhesive tabs/glue）

   耐用性：
   - [ ] 抗折性（弯曲测试）
   - [ ] 耐水性（浸泡30分钟）
   - [ ] 48小时佩戴测试

   包装：
   - [ ] 盒子材质（建议：硬纸盒 > 塑料盒）
   - [ ] 印刷质量
   - [ ] 内托设计（防压碎）
   ```

3. **打样成本预算**：
   - 样品费：$50-150/设计/工厂
   - 国际快递：$30-50/包裹
   - 总计：约$500-800（3家工厂 x 2设计）

**并行任务：设计储备**
- 在Canva/Pinterest搭建mood board
- 研究2024-2025流行趋势（参考Glamnetic新品发布节奏）
- 准备10-15个设计概念（为后续选品做准备）

## ⚠️ 常见坑点提醒

1. **不要被低价诱惑**：
   - $1-2/套的价格通常质量堪忧
   - Glamnetic级别的产品成本至少$3-4/套（FOB）

2. **一定要打样**：
   - 即使供应商案例再好看，也要实测
   - 照片和实物差距可能巨大

3. **尺寸是关键**：
   - 中国工厂的尺寸曲线可能不适合美国人手型
   - 必须提供specific sizing guide

4. **粘合系统决定复购**：
   - 劣质adhesive = 差评 = 品牌死亡
   - 建议：让供应商提供3种adhesive方案测试

5. **包装不能省**：
   - Press-on nails的"开箱体验"直接影响社交分享
   - Glamnetic的成功很大程度因为包装够"上镜"
