# 竞品分析执行指南（7天实战版）

## 目标

通过7天的系统化竞品研究，找到：
1. ✅ 市场空白点（哪里竞争少、需求大）

3. ✅ 定价策略（sweet spot价格带）
4. ✅ 营销套路（什么内容转化高）
5. ✅ 产品标准（对标Glamnetic的具体指标）

| **说明书清晰度** | | | | 10=一看就懂，有图示 |

**详细评分指南**：

```
光泽度评分参考：
10分：像Glamnetic一样的high-gloss finish，可以看到反光
8-9分：有光泽但不够镜面
6-7分：半哑光
4-5分：明显哑光
1-3分：粗糙无光

边缘打磨评分：
- 用手指沿着甲片边缘划一圈
- 10分：完全感觉不到任何毛刺或尖锐处
- 每发现一处明显毛刺 -1分

C-curve自然度评分：
- 将甲片放在自己指甲上（不粘）
- 10分：完美贴合，无gap
- 8-9分：轻微gap但可接受
- 6-7分：明显gap，需要按压
- <6分：完全不贴合，会翘起
```

## 💡 Pro Tips

### 1. 不要陷入"分析瘫痪"
- 7天必须完成，不要追求完美
- 80%的insights来自20%的分析

### 2. 用户评论>官方营销
- 差评比好评更有价值
- 1条差评 = 10个改进机会

### 3. 找"边缘案例"
- "这个产品不适合我因为______"
- 这些人可能是你的目标客户！

### 4. 数字化一切
- 拍照、录视频、截图
- 6个月后你会感谢自己

### 5. 实测>一切
- 不要相信品牌宣称
- 自己戴48小时才知道真相
