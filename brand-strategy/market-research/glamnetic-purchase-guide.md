# Glamnetic 购买指南 - 3款精选策略

## 🎯 购买目标

选择3款Glamnetic样品，覆盖：
1. **Nude/Natural款**（测试基础材料质量和自然外观）
2. **French款**（测试双色工艺精度和经典设计）
3. **Artistic/Bold款**（了解他们的设计能力和市场偏好）

预算：$50-65（含shipping）

### Step 2: 选择Nude/Natural款（第1款）

#### 🎯 选择标准

**为什么选nude款？**
- 测试基础材料质量（无法用设计掩盖瑕疵）
- 颜色均匀度一目了然
- 市场需求最大（nude系列占40%销量）
- 容易与其他品牌对比

#### 📋 具体选择方法

**在产品列表中寻找这些名字**（Glamnetic常见nude系列）：

**Top推荐（选其中1款）**：

1. **"Latte" 或 "Chai"**
   - 特征：奶茶色nude，中长杏仁形
   - 为什么选：最经典的nude色调，接近Glamnetic的signature风格
   - 价格预期：$16-20

2. **"Bare" 或 "Barely There"**
   - 特征：自然裸粉色，偏透明
   - 为什么选：测试材料透明度和自然感
   - 价格预期：$16-20

3. **"Rose Gold" 或 "Blush"**
   - 特征：带微微粉调的nude
   - 为什么选：测试subtle color的呈现能力
   - 价格预期：$16-20

#### ✅ 选择决策流程

```
打开每个nude款的产品页面，快速检查：

1. 评论数：______ 条（选择>100条的）
2. 评分：______ 星（选择≥4.5星的）
3. 看产品图片：
   - [ ] 颜色看起来自然吗？
   - [ ] 光泽度如何？（应该有高光泽）
   - [ ] 有没有lifestyle照片（实际佩戴效果）？

4. 快速扫一眼评论（不用深读）：
   - Top好评说什么？
   - 有没有差评提到颜色/质量问题？

5. 如果以上都满意 → 加入购物车！
```

#### 🔍 如果找不到上述款式

**备选策略**：
- 在筛选器选择："Nude" 或 "Natural" 类别
- 按"Best Selling"排序
- **选第1-3名中的任意一款**

### Step 3: 选择French款（第2款）

#### 🎯 选择标准

**为什么选French款？**
- 测试双色工艺（白色tip + nude base）
- 测试线条精度（French tip的边缘是否平整）
- 经典设计，永不过时
- 市场需求稳定（约占20-30%销量）

#### 📋 具体选择方法

**在产品列表中寻找这些关键词**：
- "French"
- "Classic"
- "Tips"

**Top推荐（选其中1款）**：

1. **"Classic French" 或 "Traditional French"**
   - 特征：传统白色tip，nude或透明base
   - 为什么选：行业标准款，最易对比
   - 价格预期：$16-20

2. **"Minimal French" 或 "Thin Line French"**
   - 特征：细线法式，更现代
   - 为什么选：测试精细工艺能力
   - 价格预期：$16-20

3. **"Ombré French" 或 "Gradient French"**
   - 特征：渐变法式
   - 为什么选：测试渐变工艺（更高难度）
   - 价格预期：$18-22

#### ✅ 选择决策流程

```
1. 打开French款产品页面

2. 重点观察产品图片：
   - [ ] French tip的边缘是否笔直？（放大看）
   - [ ] 左右两片是否对称？
   - [ ] 白色tip的宽度是否一致？

3. 查看评论中的用户上传照片（非常重要！）：
   - 实际佩戴效果vs官方图片差距大吗？
   - 用户照片中French line是否整齐？

4. 满意 → 加入购物车
```

### Step 4: 选择Artistic/Bold款（第3款）

#### 🎯 选择标准

**为什么选artistic款？**
- 了解Glamnetic的设计风格偏好
- 测试复杂设计的质量（印刷清晰度、装饰牢固度）
- 研究什么风格在市场上受欢迎
- 为你的首批SKU设计提供灵感

#### 📋 具体选择方法

**策略A：选当前bestseller（推荐✅）**

```
1. 在"Best Sellers"页面
2. 排除nude和French款
3. 剩下的就是artistic款
4. 选评论数最多的那款

为什么？
→ 评论多=卖得好=市场验证过的设计
→ 你能学到"什么设计受欢迎"
```

**策略B：选你个人喜欢的设计**

```
如果没有特别突出的bestseller，选择：
1. 你觉得最漂亮的
2. 或最unique的
3. 或你觉得"我也想做这种风格"的

为什么？
→ 你的审美很重要
→ 如果你自己都不喜欢，很难有激情去卖
```

#### 🎨 Glamnetic常见Artistic风格

**可能看到的设计类型**：

1. **Glitter/Sparkle系列**
   - 特征：闪片、亮粉
   - 例如："Starlight", "Diamond", "Shimmer"
   - 测试重点：闪片是否会掉落？

2. **Marble/Abstract系列**
   - 特征：大理石纹、抽象图案
   - 例如："Marble", "Swirl", "Abstract"
   - 测试重点：图案是否清晰？左右是否对称？

3. **Animal Print系列**
   - 特征：豹纹、斑马纹、蛇纹
   - 例如："Leopard", "Python", "Zebra"
   - 测试重点：印刷精度

4. **Seasonal/Holiday系列**
   - 特征：节日主题（Valentine's, Christmas等）
   - 如果现在有seasonal款，可以考虑
   - 测试重点：装饰是否牢固？

5. **3D/Embellished系列**
   - 特征：带立体装饰（rhinestones, studs）
   - 价格通常更高（$20-25）
   - 测试重点：装饰是否容易脱落？日常活动是否会钩到东西？

## ❓ 常见问题

### Q1: 如果我最喜欢的款式没货怎么办？
**A**: 选择同类别的次优选项，或者：
- 加入waitlist（如果有）
- 先买其他两类，等补货再买

### Q2: 如果3款都差不多价格，我应该买贵的还是便宜的？
**A**:
- **买bestseller** > 买贵的或便宜的
- 评论数多=市场验证=更有研究价值

### Q3: 我应该买短款还是长款？
**A**:
- **买中长款**（Medium或Long）
- 原因：更能测试佩戴舒适度和持久度
- 短款测试价值较低

### Q4: 看到"New Arrival"，要买新款吗？
**A**:
- **第一次买不要买新款**
- 原因：新款评论少，无法判断市场反应
- 买成熟款（>100 reviews）更有研究价值

### Q5: Glamnetic vs Static Nails，应该先买哪个？
**A**:
- **先买Glamnetic**（今天）
- Static Nails可以明天或后天买
- 原因：Glamnetic是你的主要对标，优先研究
